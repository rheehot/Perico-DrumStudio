## Development Server Run
- 1 `yarn install` 프로젝트에 필요한 패키지 설치
- 2 `yarn start` 개발 서버 실행

## Deploy
- 1 `yarn build` 소스 파일 최적화
- 2 `yarn deploy` 소스 파일 S3 업로드
- 3 `yarn invalidate` CDN 새로고침
